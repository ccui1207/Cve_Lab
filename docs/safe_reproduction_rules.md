# 安全复现规则

## 允许做的事情

- 在本地模拟器中复现公开 CVE。
- 在本人拥有的测试手机中复现公开 CVE。
- 分析官方补丁。
- 记录复现环境、日志和截图。
- 验证修复前后行为差异。
- 编写防御和修复建议。

## 不做的事情

- 不攻击真实线上目标。
- 不扫描他人设备。
- 不批量利用漏洞。
- 不编写持久化控制逻辑。
- 不绕过检测系统。
- 不窃取数据。
- 不将复现代码用于未授权环境。

## PoC 运行前检查

运行任何公开 PoC 前，先检查：

```text
是否联网
是否下载未知二进制
是否删除文件
是否修改系统分区
是否请求 root
是否包含混淆代码
是否上传设备信息
是否存在恶意 payload
```

发现可疑行为时，不运行。
